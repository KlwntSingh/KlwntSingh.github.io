# CCNA Security

## planes 
* #### management plane 
	* managing the networking devices 
	* ssh, telnet 
* #### control plane 
	* anything that involves the cpu of networking device 
	* routing protocol, traffic destined for network devices 
* #### data plane 
	* traffic forwarded by networking device
